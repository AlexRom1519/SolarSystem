Реализация трёхмерной карты солнечной системы.

Управление:

мышь - изменение взгляда

WASD - движение вперёд, влево, назад, вправо

Q - движение камеры вниз
E - движение камеры вверх

F - увеличение радиуса планет в схематичном режиме
G - уменьшение радиуса планет в схематичном режиме

R - уменьшение радиуса орбит в любом режиме
T - увеличение радиуса орбит в любом режиме

X - увеличение скорости
C - уменьшение скорости

Пробел - остановка движения
TAB - смена режима между схематичным и реальным

У всех изменений есть пределы, в случае их достижения об этом пишется во втором консольном окне.

Дополнительно:

Камера двигается не линейно, притормаживая около планет
Есть сферическая карта звёздного неба
Есть возможность настроить скорость анимации (X / C)
Нарисованы орбиты линиями
У сатурна реализованы полурозрачные кольца при помощи одной текстуры, (текстура вращается вместе с планетой)

Бинарный файл формируется в папке bin.
